# Update-TriliumNoteOrder

## SYNOPSIS

Updates the order of notes under a specific parent note.

## SYNTAX

### __AllParameterSets

```
Update-TriliumNoteOrder [-ParentNoteId] <string> [-SkipCertCheck] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ALIASES

This cmdlet has no aliases.

## DESCRIPTION

This function updates the order of notes under a specific parent note based on the provided parent note ID.

## EXAMPLES

### EXAMPLE 1

```powershell
Update-TriliumNoteOrder -ParentNoteId "root"
```

Updates the order of child notes under the root note.
